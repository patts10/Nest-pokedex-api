# Run in development environment

1. Clone repository
2. Run
    
   ```
   npm install
   ```
3. Must have Nest CLI installed
   
   ```
   npm i -g @nestjs/cli
   ```
4. Build DB
   
   ```
   docker-compose up -d
   ```

   ## Stack used
   * MongoDB
   * Nest
